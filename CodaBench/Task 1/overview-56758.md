<h1>AraSentEval 2026: Subtask 1 - Arabic Dialect Sentiment Analysis</h1>
<p>This subtask is a multi-class classification task focused on identifying the sentiment (positive, negative, or neutral) of texts written in various Arabic dialects.</p>
<p>It emphasizes the need for robust models that can handle the lexical and syntactic diversity of the Arab world.</p>
<h2>Task Description</h2>
<ul>
    <li><strong>Input</strong>: An Arabic sentence written in a specific dialect.</li>
    <li><strong>Output</strong>: A sentiment label from the set {positive, negative, neutral}.</li>
</ul>
